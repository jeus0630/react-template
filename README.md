# Vite React Template! 
A working-level react template based on vite bundling tool.

## Installation and Usage
1. Install libraries using yarn
    ```zsh
    yarn
    ```
2. Install Husky 
    ```zsh
    yarn husky install
    ```
3. Create env
    - copy&paste `.env.default` and rename it to `.env`
4. Enjoy your coding!
